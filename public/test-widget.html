Upload Widget Test

Testing element detection for div and input elements

Test 1: Div Element

Using <div flow-upload="widget" flow-name="attachments">
Expected textarea name: attachments

Test 2: Input Element (Auto Name)

Using <input type="file" name="documents" flow-upload="widget">
Expected textarea name: documents (from input's name attribute)

Test 3: Input Element (Name Override)

Using <input name="old_name" flow-upload="widget" flow-name="new_name">
Expected textarea name: new_name (flow-name overrides input name)

Test 4: Input Element (Default Name)

Using <input type="file" flow-upload="widget"> (no name attribute)
Expected textarea name: uploaded_files (default)
